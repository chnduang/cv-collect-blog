# setTimeout第三个参数

> [window.setTimeout - Web API 接口参考 | MDN (mozilla.org)](https://developer.mozilla.org/zh-CN/docs/Web/API/setTimeout)



```js
setTimeout((i)=> {
	console.log(i, j)
},20, 'a','b','c')

// output: 'a','b'
```
